Project «CoTe»

Страница проекта - CoTe

Сайт предназначен для форматирования кода и приведения его в читаемый вид.
Форматирование происходит по символам «{», «}», и «;», которые являются неотъемлемой частью многих ЯП.
На данный момент на сайте доступна только настройка параметра отступа.
(В скором времени ожидаются обновления с добавлением настроек форматирования)

Удобство

Простая до безумия схема использования сайта.

Лаконичный дизайн

Ничего лишнего. Максимальное воспевание красоты аскетизма.

Многоязычное редактирование

Редактирование кода «не зависит от языка программирования».

Примеры использования

На сайте имеются примеры для ознакомления с возможностями работы программы.

Широкая ца

Сайт будет полезен как для студентов, так и для любого человека занимающегося программированием.

Поддержка

Обновления
